# lucidlink-assignment

## Semantic versioning for this package: 
This package uses semantic versioning. To bump the version you need to add +semver:<patch|minor|major> to the PR name. 
For example if the current version is 1.0.1 and you create a PR with title "My perfect PR +semver:minor" it will bump the version to 1.1.0. If no +semver is provided the PR will be treated as a patch PR.
